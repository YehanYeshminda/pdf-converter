About Unix Timestamps

  • Unix Timestamp: Number of seconds (or milliseconds) that have elapsed since January 1, 1970 00:00:00 UTC (Unix Epoch)
  • Seconds: Standard Unix timestamp format used in most programming languages
  • Milliseconds: Used in JavaScript and some other languages
  • UTC: Coordinated Universal Time - the primary time standard
  • ISO 8601: International standard for date and time representation
